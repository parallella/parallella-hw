Parallella 
=================================

## Summary

* Credit card sized Single Board Computer
* Adapteva 16-core Epiphany RISC 
* Xilinx Zynq FPGA 
* 1GB DDR3 RAM
* Gigabit Ethernet, HDMI, USB 2.0, micro-SD
* 48 GPIO pins
* 96 elink pins
* <5W peak power
* 3.275" x 2.150"
